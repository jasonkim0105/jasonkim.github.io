Deecord

A fullstack project using Ruby on Rails for the backend and React/Redux for the frontend. A clone of the popular instant messaging app Discord, users are able to create servers and channels to message each other in live time. In addition to channel-based messaging, users are able to direct message others.

Plant Press

Plant-Press is a mobile tracking app that allows plant owners to post their plants and record any reminders neccesary for each plant's care needs. At the same time, plant owners can showcase their plant's progress to their friends through the posts feature and follow their friend's plants to gain updates as their plants grow! Created with React/Reudx, MongoDB, Express.js, AWS S3 and Node.js

Wave-Watch

Wave-Watch is an app that displays the surf conditions of popular surf sites along the coast of California. Multiple locations can be compared simultaneously. Built using vanilla JS and compiled with Webpack. An external API was used for the surf conditions.
